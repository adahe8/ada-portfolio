Reasons to dislike React.

October 20, 2025

Lately, I've been thinking more about the tradeoffs of modern frameworks that are quite popular. Don't get me wrong, I've liked using React! It lets me write html and JS together, and I find using hooks and React components quite fun. Through coursework and craft notes from professionals I admire though, I've been recognizing more and more the pain points of this framework. A hilarious short video from the content creator fireship, summarizes those issues quite well.
